# Hive Corps Demo Script

Target length: 2-3 minutes.

## 0:00-0:20 - Problem

"B2B quote teams lose revenue because customer requests are messy, pricing checks are manual, and customer-specific knowledge lives in people's heads. Hive Corps solves this with a self-improving agent corps for quote operations."

## 0:20-0:40 - Product

Open the dashboard.

"This is Hive Corps. It is submitted under Track 4, Autopilot Agent. The system runs a complete workflow from inbound request to quote generation, human approval, output, audit log, and learning."

## 0:40-1:20 - First Run

Click **Run quote workflow**.

Show:

- Inbound request from AdeptWorks Lagos.
- Intake Agent extracts quantity, location, urgency, and laptop need.
- Memory Agent recalls Dell preference, conservative delivery preference, and support preference.
- Catalog Agent selects Dell Latitude plus 3-year onsite support.
- Risk Agent flags the quote because the proposed discount exceeds policy and delivery is too aggressive.

Key line:

"The agent is autonomous, but not unchecked. Risky decisions stop at the approval gate."

## 1:20-1:50 - Approval + Output

Show the generated quote and approval state.

Explain the human edit:

"The human changes the discount from 15% to 8% and delivery from 4 days to 6 business days. That produces an approved quote and customer response."

## 1:50-2:20 - Self-Improvement

Show Learning Agent proposals.

"Hive Corps does not silently rewrite itself. It proposes controlled memory and policy updates: cap AdeptWorks discounts at 8%, use conservative delivery for 20+ devices, and keep recommending Dell with 3-year support."

## 2:20-2:45 - Second Improved Run

Click **Run learned second quote**.

Show:

- No manual discount correction.
- Learned customer preference applied.
- Safer approval state.

Key line:

"This is the core differentiator: act, review, learn, and act better."

## 2:45-3:00 - Proof

Open proof links:

- Latest agent run JSON
- Generated quote JSON
- Audit log JSON
- Alibaba Cloud deployment proof plan

Close:

"Hive Corps shows how Qwen Cloud can power production agents that complete real business workflows, remain auditable, and improve from human feedback."
